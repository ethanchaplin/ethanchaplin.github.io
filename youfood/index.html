Design Requirements

I identified the following as design goals for the interface

  • Be able to upload audio files to listen to later
  • Be able to add or remove goals to be checked off daily
  • Set goals for a daily average of practice minutes
  • Flexible layout for a customizable view
  • The ability to track the same song over the course of several days/weeks

Implementation Details

This project was built with Svelte. No external CSS or component libraries were used, all was built from scratch. Each element was a component, except for the sidebar and the navbar, which I kept within App.svelte.

Future Work
